3. QUY TRÌNH CÀI ĐẶT (7 BƯỚC)

BƯỚC 1: Cài Node.js

- Tải: https://nodejs.org/
- Kiểm tra: node --version

BƯỚC 2: Cài MySQL

- Cài MySQL Server
- Ghi nhớ password root

BƯỚC 3: Cài Dependencies

npm install express mysql2 cors

BƯỚC 4: Cấu hình Password

⚠️ ĐỔI PASSWORD trong:
- app.js (dòng 16)
- setup_database.js (dòng 6)

BƯỚC 5: Khởi tạo Database

node setup_database.js

BƯỚC 6: Chạy Server

node app.js

BƯỚC 7: Truy cập

http://localhost:3000
